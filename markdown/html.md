# [List](/html/list.html)

### 3 Types of List, List Tags & Attributes
* `ul` - unordered list
    * `li` - list item
* `ol` - ordered list
    * `li` - list item
        * type
            * `type="1"` - number
            * `type="A"` - uppercase
            * `type="a"` - lowercase
            * `type="I"` - uppercase roman number
            * `type="i"` - lowercase roman number
        * `reversed` - reversed list
        * control list
            * `start=" "`
            * `value=" "`
* `dl` - description list
    * `dt` - term
    * `dd` - details the term
        * `list style types` - `disc`, `circle`, `square`, `none`
        * `compact` - smaller list (not supported by html5)
